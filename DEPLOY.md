# Deploy to Vercel

This folder is ready for Vercel as a Next.js project.

The runtime is pinned to Node.js 22 and the dependency lockfile is validated with a clean install.

1. Unzip the portfolio.
2. Open a terminal in the unzipped folder.
3. Run `npm install`.
4. Run `npm run build` to verify it.
5. Run `npx vercel` and follow the prompts, or push the folder to a Git repository and import it from the Vercel dashboard.

No environment variables are required.
